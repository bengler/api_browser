GET /api/v1/events/latest


Returns a list of events most recently updated. The body for the event and the venue is Textilized by default.

Parameter Type Description
since Optional datetime The time you want changes since. JSON-formatted JavaScript Date object.
page Optional integer The page of results you want returned. Default is 1.
