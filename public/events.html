GET /api/v1/users/:id/events


Returns events that the user has a participation on, including attending performing recommending Sorted by time descending, limited to 30 events per page. The body for the event and the venue is Textilized by default.

Parameter Type Description
id Required user id

username

'current'

The id of the user
page Optional integer The page of results you want returned. Default is 1
