POST /api/v1/events/:id/rsvp


Create or remove a participation on an instance of an event. This will also follow the event on Underskog if the kind is any other than none and the user has not previously unfollowed it.

Parameter Type Description
id Required integer

The id of the event
kind Required 'attend

'perform'

'recommend'

'none'

The type of participation to set. When none any existing RSVP will be removed.
